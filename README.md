# 🌈 나의 감정 날씨 일기

> 한국 초등학생을 위한 인터랙티브 감정 일기 | Interactive emotion diary for Korean elementary students using weather metaphors

## 📌 소개

**나의 감정 날씨 일기**는 초등학생들이 자신의 감정을 날씨에 비유하여 표현하고, 신체 감각과 연결하여 이해할 수 있도록 돕는 웹 기반 교육 도구입니다. 아이들이 감정을 인식하고 표현하는 능력을 키우며, 정서적 자기조절 능력을 향상시킬 수 있습니다.

### 🎯 주요 특징
- 🌦️ **8가지 날씨 메타포**로 감정 표현
- 🧘 **신체 감각과 감정 연결** 학습
- 💌 **날씨별 위로 메시지** (24개의 맞춤형 문장)
- 🎨 **위로 카드 생성** (3가지 디자인 템플릿)
- 💾 **자동저장** 및 **고품질 이미지/PDF 다운로드**
- 📱 **반응형 디자인**으로 모바일/태블릿 완벽 지원
- ♿ **웹 접근성** 준수 (스크린리더, 키보드 네비게이션)

## 📖 사용 방법

### 1. 기본 정보 입력
- 날짜 선택
- 학년/반 입력
- 이름 입력

### 2. 감정 날씨 선택
8가지 날씨 중 현재 감정과 가장 비슷한 날씨를 선택합니다:

| 날씨 | 감정 상태 |
|------|----------|
| ☀️ 맑음 | 기쁨, 즐거움, 행복 |
| ☁️ 흐림 | 무덤덤, 심심함 |
| 🌧️ 비 | 슬픔, 우울함 |
| ⛈️ 폭풍 | 화남, 짜증, 분노 |
| ❄️ 눈 | 차분함, 평온함 |
| 🌈 무지개 | 희망, 기대, 설렘 |
| 🌬️ 바람 | 불안, 초조함 |
| 🌫️ 안개 | 혼란, 막막함 |

### 3. 신체 변화 기록
- 인터랙티브 신체 다이어그램에서 변화가 있는 부위 클릭
- 각 부위별 구체적인 증상 기록

### 4. 감정의 원인 탐색
- 무슨 일이 있었는지
- 그때 어떤 생각이 들었는지
- 다음에는 어떻게 하고 싶은지

### 5. 나에게 주는 위로 메시지 (신규 v3.0)
- 날씨별 맞춤형 위로 문장 3개 제공 (총 24개)
- 제공된 문장 선택 또는 직접 작성 가능
- 또래교과 자기위로 언어 기반

### 6. 위로 카드 만들기 (신규 v3.1)
- 💌 **이름 입력**: 카드에 들어갈 이름
- 🎨 **템플릿 선택**: 3가지 감성 디자인 중 선택
- 🖼️ **고품질 다운로드**: PNG (2400×3600px) 또는 PDF (A4)
- 📱 **실시간 미리보기**: 선택한 디자인 즉시 확인

### 7. 저장 및 공유
- 📸 **전체 일기 저장**: PNG 파일로 다운로드
- 💌 **위로 카드 저장**: 개인 맞춤형 카드 이미지/PDF
- 🖨️ **인쇄**: 2페이지 형식으로 인쇄
- 🔄 **다시 작성**: 새로운 일기 작성

## 🏫 교육 활용 방안

### 수업 활용
- **아침 활동**: 하루를 시작하며 감정 체크인
- **종례 시간**: 하루를 마무리하며 감정 정리
- **상담 활동**: 개별 상담 시 감정 탐색 도구로 활용
- **SEL 프로그램**: 사회정서학습 프로그램과 연계

### 기대 효과
1. **감정 인식 능력 향상**: 자신의 감정을 구체적으로 인식
2. **감정 표현 능력 발달**: 다양한 방식으로 감정 표현
3. **신체 감각 인식**: 감정과 신체 반응의 연결 이해
4. **메타인지 발달**: 감정의 원인과 결과 분석
5. **자기 조절 능력**: 향후 대처 방안 계획
6. **자기위로 능력**: 스스로를 격려하고 위로하는 힘 기르기 (v3.0+)
7. **정서적 회복력**: 힘든 순간에 꺼내볼 수 있는 마음의 부적 (v3.1+)

## 📱 주요 기능 상세

### 자동저장
- 모든 입력은 1초 후 자동으로 브라우저에 저장
- 페이지 새로고침 시 자동으로 복원
- 저장 완료 시 상단에 인디케이터 표시

### 접근성
- 스크린리더 완벽 지원 (ARIA 레이블)
- 키보드 네비게이션 (Tab, 방향키, Enter)
- 고대비 모드 지원
- 모바일 터치 영역 최적화 (최소 48px)

### 인쇄 최적화
- A4 용지 2페이지 자동 분할
- 불필요한 요소 자동 제거
- 입력 내용 깔끔한 텍스트로 변환

### 파일명 규칙
이미지 저장 시 체계적인 파일명 자동 생성:
- **전체 일기**: `감정날씨일기_YYYYMMDD_학년반_이름.png`
- **위로 카드**: `위로카드_YYYYMMDD_이름.png` 또는 `.pdf`
- 예시: `감정날씨일기_20250812_5학년2반_홍길동.png`

### 위로 카드 템플릿 (v3.1)
3가지 감성 디자인:
1. **☀️ 따뜻한 햇살**: 밝고 희망적인 노란색 그라데이션
2. **🌙 차분한 밤하늘**: 고요하고 안정적인 남색 그라데이션
3. **🌲 싱그러운 숲**: 생동감 넘치는 초록색 그라데이션

각 카드 크기: 800×1200px (4:6 비율), 고해상도 렌더링 2400×3600px
